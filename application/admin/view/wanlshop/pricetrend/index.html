数据指标说明
平台均价
AVG(平台SKU价格)
统计 shop_id = 1（平台概念店）的所有在售SKU价格的算术平均值
示例：平台有3个SKU，价格分别为100、120、110，则平台均价 = (100+120+110)/3 = 110
商家均价
AVG(商家SKU价格)
统计 shop_id ≠ 1（入驻商家）的所有在售SKU价格的算术平均值
示例：商家有4个SKU，价格分别为95、100、105、100，则商家均价 = (95+100+105+100)/4 = 100
价差比例
(平台均价 - 商家均价) / 商家均价 × 100%
平台价格相对于商家价格的溢价/折价比例，正数表示平台价格高于商家，负数表示低于商家
示例：平台均价110，商家均价100，则价差比例 = (110-100)/100 × 100% = +10%
商家最低/最高
MIN(商家SKU价格) / MAX(商家SKU价格)
所有入驻商家在售SKU中的最低价格和最高价格
示例：商家SKU价格为95、100、105、100，则最低=95，最高=105
数据覆盖
COUNT(DISTINCT category_id) / COUNT(sku_id)
统计有在售SKU的商品分类数量，以及所有在售SKU的总数量
示例：玉米分类有3个SKU，大豆分类有2个SKU，则覆盖2个分类，共5个SKU
入驻商家
COUNT(DISTINCT shop_id) WHERE shop_id ≠ 1
统计有在售SKU的入驻商家数量（不含平台概念店）
示例：3家商家各有商品在售，则入驻商家 = 3
